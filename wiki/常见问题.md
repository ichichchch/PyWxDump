# 常见问题

本文档收录使用 PyWxDump 时的常见问题及解决方案。

---

## 安装问题

### Q: 怎么下载/怎么安装？

**方法一：下载可执行文件**
- 进入 [Releases](https://github.com/xaoyaoo/PyWxDump/releases) 下载最新版 exe 文件

**方法二：pip 安装**
```bash
pip install PyWxDump
```

### Q: 命令未找到 / 命令无法识别

1. 检查是否使用了中文输入法，请切换为英文输入法
2. 确认输入的是 `wxdump.exe info` 而不是只输入 `info`
3. 确认当前目录是 exe 文件所在目录
4. 如果还是无法识别，将 exe 文件拖到命令行窗口，再输入命令

### Q: 刚打开就闪退

1. 请使用 cmd 或 PowerShell 打开，不要直接双击 exe 文件
2. 如果使用 pip 安装，检查 Python 环境是否正确
3. 检查 Python Scripts 目录是否添加到环境变量

### Q: 模块导入失败

```bash
pip install -r requirements.txt
```

## 使用问题

### Q: 如何使用？

1. 打开微信电脑版并登录
2. 进入 exe 文件所在目录
3. 按住 Shift + 右键，选择"在此处打开命令窗口"
4. 输入 `./wxdump.exe` 或 `wxdump`（pip 安装）
5. 根据提示操作

### Q: 不登录微信能获取密钥吗？

**不能**。必须登录微信才能获取密钥。但可以保存密钥，下次使用时无需登录。

### Q: 微信锁定情况下能获取密钥吗？

**可以**。只要微信进程在运行，即使锁定也可以获取密钥。

### Q: 每台电脑的密钥会变化吗？

同一设备、同一微信号、不删除数据的情况下，**密钥保持不变**。

### Q: 可以查看别人的聊天记录吗？

**不可以**。只能查看有密钥的数据库。如果你有别人的密钥和数据库，理论上可以解密，但这是**违法行为**。

## 数据库问题

### Q: 微信数据库在哪里？

1. 打开微信电脑版并登录
2. 设置 → 文件管理 → 打开文件夹
3. 进入 `MSG` 文件夹

完整路径通常为：
```
C:\Users\用户名\Documents\WeChat Files\wxid_xxx\MSG\
```

### Q: 解密失败的原因？

| 原因 | 说明 |
|------|------|
| 密钥不匹配 | 使用了其他账号的密钥解密 |
| 数据库未加密 | 部分数据库本身就是未加密的 |
| 文件损坏 | 数据库文件已损坏 |

### Q: decrypted 文件夹里的内容是什么？

| 路径 | 说明 |
|------|------|
| `decrypted/MicroMsg` | 联系人数据库 |
| `decrypted/Multi/MSG0-5` | 聊天记录数据库 |
| `decrypted/Multi/MediaMSG0-5` | 媒体消息数据库 |

### Q: MSG0-MSG5 应该用哪个？

- `MSG0` - 最旧的聊天记录
- `MSG5` - 最新的聊天记录
- 建议使用 `merge` 命令合并所有数据库

### Q: 如何合并数据库？

```bash
wxdump merge -i "MSG0.db,MSG1.db,MSG2.db" -o "output/merge.db"
```

## 版本兼容问题

### Q: 微信版本差异

| 版本 | 说明 |
|------|------|
| < 3.7.0.30 | 不登录可获取部分信息，登录后可获取密钥 |
| > 3.7.0.30 | 必须登录才能获取信息和密钥 |

### Q: 系统要求

- Windows 10 64位及以上
- Python 3.8 及以上
- 其他版本可能有兼容性问题
