# 快速开始

本指南帮助您快速上手使用 PyWxDump。

## 小白教程

### 步骤 1：准备工作

1. 打开微信电脑版并登录
2. 下载 [wx_dump.exe](https://github.com/xaoyaoo/PyWxDump/releases)

### 步骤 2：运行程序

1. 双击 `wx_dump.exe` 运行
2. 程序会自动打开浏览器
3. 访问 http://127.0.0.1:5000/

### 步骤 3：使用界面

根据网页界面的提示进行操作即可。

---

## 详细教程

### 一键启动 Web 界面

```bash
wxdump ui
```

这将：
- 自动获取微信信息
- 启动本地服务器（端口 5000）
- 自动打开浏览器

### 完整使用流程

#### 1. 获取微信信息

```bash
wxdump info
```

输出示例：
```
================================
[+]      pid: 12345
[+]  version: 3.9.8.25
[+]  account: your_account
[+]   mobile: 138****1234
[+] nickname: 昵称
[+]     mail: email@example.com
[+]     wxid: wxid_xxxxxxxx
[+]      key: xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
[+]   wx_dir: C:\Users\xxx\Documents\WeChat Files\wxid_xxxxxxxx
================================
```

#### 2. 获取数据库路径

```bash
wxdump wx_path
```

#### 3. 解密数据库

```bash
wxdump decrypt -k <密钥> -i <数据库路径> -o <输出路径>
```

示例：
```bash
wxdump decrypt -k "你的64位密钥" -i "C:\Users\xxx\Documents\WeChat Files\wxid_xxx\MSG" -o "C:\decrypted"
```

#### 4. 合并数据库

```bash
wxdump merge -i "数据库1.db,数据库2.db" -o "输出路径"
```

#### 5. 查看聊天记录

```bash
wxdump dbshow -merge "合并后的数据库路径"
```

### 使用 Python API

```python
from pywxdump import *

# 获取微信信息
wx_info = get_wx_info(WX_OFFS, is_print=True)

# 解密数据库
result = batch_decrypt(
    key="你的64位密钥",
    db_path="数据库路径",
    out_path="输出路径"
)

# 启动服务器
start_server(port=5000)
```

## 使用场景

| 场景 | 推荐命令 |
|------|----------|
| 快速查看聊天记录 | `wxdump ui` |
| 仅获取密钥 | `wxdump info` |
| 批量解密数据库 | `wxdump decrypt` |
| API 开发 | `wxdump api` |

## 注意事项

1. **必须先登录微信**才能获取密钥
2. 密钥与设备和账号绑定，同一设备同一账号密钥不变
3. 解密失败可能因为：
   - 密钥不匹配（非当前登录账号的数据库）
   - 数据库本身未加密
4. 微信锁定状态下也可以获取密钥
