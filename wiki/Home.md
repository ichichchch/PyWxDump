# PyWxDump 文档

欢迎使用 **PyWxDump** 文档！本文档提供了使用 PyWxDump 提取和管理微信数据的完整指南。

## 项目简介

PyWxDump 是一款功能强大的 Python 工具，用于：
- 获取微信账号信息（昵称、账号、手机号、邮箱、数据库密钥）
- 解密微信数据库
- 通过 Web 界面查看聊天记录
- 导出聊天记录为 HTML、CSV、JSON 格式
- 备份微信数据

**当前版本：** 3.1.45

## 核心功能

| 功能 | 说明 |
|------|------|
| **多账号支持** | 支持管理多个微信账号的数据 |
| **版本兼容** | 支持所有微信版本 |
| **数据导出** | 支持导出为 CSV、HTML、JSON 格式 |
| **Web 界面** | 内置网页界面查看聊天记录 |
| **API 服务** | 基于 FastAPI 的 REST API |
| **数据库合并** | 合并多个数据库文件 |
